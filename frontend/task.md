# 任务：前端界面重构

## 任务描述
重构 `med-platfrom` 前端，使其界面不仅现代优美，而且具备优雅的动效。主要对标 `other-platfrom` 的设计风格，实现响应式布局（自动适配 PC 和移动端）以及全新的左侧边栏导航架构。

## 已完成事项
- [x] **布局重构 (`MainLayout`)**: 
    - 实现左侧固定侧边栏 (Desktop) 与 顶部导航+抽屉菜单 (Mobile)。
    - 采用 Flex 布局与 CSS 变量管理配色 (参考了 slate-50, blue-600 等色调)。
    - 添加页面切换淡入淡出动效。
- [x] **路由配置**: 
    - 更新 `router/index.ts`，为核心功能页面应用新的 `MainLayout`。
- [x] **页面适配**:
    - **首页 (`Home.vue`)**: 移除顶部 Header，重组头部控制栏，适配 Flex 高度。
    - **错题本 (`Mistakes.vue`)**: 统一头部 UI，卡片化容器风格，修复高度溢出问题。
    - **收藏夹 (`Favorites.vue`)**: 统一 UI 风格与交互布局。
    - **笔记本 (`MyNotes.vue`)**: 统一 UI 风格，移除冗余导航。
    - **个人中心 (`Profile.vue`)**: 开启垂直滚动支持，避免内容被截断。

## 备注
- 所有页面现在包含在 `MainLayout` 的内容区域中，利用 `100%` 高度和内部滚动机制，实现了类应用 (App-like) 的体验。
- 样式采用了 Vanilla CSS 结合 Naive UI 组件，未使用 Tailwind 但复刻了其视觉风格。
